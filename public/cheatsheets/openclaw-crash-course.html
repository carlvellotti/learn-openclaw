1
Install & Launch
# Install globally npm install -g @anthropic-ai/openclaw # Run the setup wizard openclaw onboard # Start the gateway openclaw start # Verify it's running openclaw status

Requires: Node.js 22+, an Anthropic API key

2
Know Your Config Files

All personality files live in ~/clawd/

SOUL.mdPersonality, tone, values
USER.mdInfo about you — name, location, prefs
MEMORY.mdLong-term facts & knowledge
RULES.mdHard rules for edge cases
MESSAGING.mdMessage routing rules
HEARTBEAT.mdProactive scheduled actions

Main config: ~/.openclaw/openclaw.json

3
Write Your SOUL.md

This is the most important file. Start minimal:

# ~/clawd/SOUL.md You are my personal assistant. Be concise and direct. No pleasantries unless I start them. Prefer bullets over paragraphs. When uncertain, ask — don't guess.

Don't rush this. Generic SOUL = generic responses. Add context about your work style over time.

4
Connect a Channel

Pick one to start. Add to openclaw.json:

// iMessage (macOS only) "channels": { "imessage": { "enabled": true, "pairedDevices": ["+15551234567"] } }
  • Slack: needs Bot Token + App Token
  • Telegram: get token from @BotFather
  • Discord: create app at developers portal
  • WebChat: built-in at localhost:18789
5
Lock Down Security

Set these before going live:

"security": { "dmPolicy": "paired-only", "groupPolicy": "allowlisted", "sandboxEnabled": true, "allowedPaths": ["~/Documents"], "maxTokensPerRequest": 4096 }
  • paired-only — only responds to your devices
  • allowlisted — no surprise group chats
  • sandbox — limits filesystem access
  • Set spending limits on your AI dashboard
6
Your First Automation

Add a cron job to openclaw.json:

"cron": [{ "schedule": "0 7 * * 1-5", "prompt": "Morning briefing: weather, calendar, top tasks, overnight alerts.", "channel": "imessage" }]
0 7 * * *Every day at 7 AM
0 7 * * 1-5Weekdays at 7 AM
*/30 * * * *Every 30 minutes
0 17 * * 5Fridays at 5 PM
0 8 1 * *1st of month, 8 AM
~
Essential Commands
openclaw start # Start gateway openclaw start --daemon # Background openclaw stop # Stop openclaw restart # Restart openclaw status # Health check openclaw doctor # Full diagnostic openclaw config # Edit config
openclaw skill list # Browse skills openclaw skill install X # Install skill openclaw channel add X # Add channel openclaw channel test X # Test channel openclaw memory list # View memory openclaw logs --tail 50 # Recent logs openclaw test model # Test AI conn
!
If Something Breaks
  • Won't start? Check lsof -i :18789 for port conflicts
  • No response? Check dmPolicy — is your device paired?
  • API errors? Run openclaw test model
  • Cron not firing? Gateway must be running + valid JSON
  • High costs? Set maxTokensPerRequest + check cron frequency
  • Config issues? Run openclaw doctor for full diagnostic
