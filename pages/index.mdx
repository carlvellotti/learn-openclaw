# Learn OpenClaw

## The free, comprehensive guide to setting up your always-on AI assistant.

![Learn OpenClaw — Installation, SOUL.md, Channels, Memory, Automation, Security, Sub-Agents, and Deployment](/images/og-image.png)

OpenClaw is the fastest-growing open source project in history — 145K+ GitHub stars and counting. But **70% of the content out there is surface-level**, and most people who install it get stuck after the initial setup.

This guide goes deep. From architecture fundamentals to advanced automation, we cover everything you need to actually make OpenClaw useful — not just installed.

---

## What's Inside

### [Getting Started](/getting-started/what-is-openclaw)
Understand what OpenClaw actually is, how to install it, and what to configure in your first 24 hours.

### [Security](/security/dm-policies)
The #1 concern for new users. A plain-English checklist covering DM policies, pairing, sandboxing, and API key management.

### [Channels](/channels/overview)
Set up iMessage, Slack, WhatsApp, Telegram, Discord, and Signal — with a comparison of when to use each.

### [Core Concepts](/core-concepts/soul-md)
Master the systems that make OpenClaw powerful: SOUL.md personality design, the memory system, skills & ClawHub, and cron automation.

### [Advanced](/advanced/sub-agents)
Go deeper with sub-agents, voice mode, deployment options, and cost optimization strategies.

### [Use Cases](/playbooks)
Real-world playbooks for personal assistant, content creation, research, daily briefings, and more.

### [OpenClaw vs Claude Code](/openclaw-vs-claude-code)
They're not competitors — they're complements. Understand when to use each and how they work together.

### [Cheat Sheet](/cheatsheet)
The quick reference you'll keep coming back to. Key commands, config files, security defaults, and troubleshooting.

## Who This Is For

This guide is for anyone who wants to use OpenClaw but feels overwhelmed by the setup process. You don't need to be a developer.

- **Technical-adjacent professionals** — PMs, marketers, founders who use AI daily
- **Tinkerers** — People who installed OpenClaw but got stuck after "Hello World"
- **Power users** — Already running OpenClaw but want to go deeper on security, memory, and automation
